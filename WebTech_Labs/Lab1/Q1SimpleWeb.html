HTML Basics

A Simple Introduction to HTML Tags and Their Uses

Use of <p> Tag

<p> Tag: The paragraph tag is used to define a paragraph of text in HTML. It's a block-level element that automatically adds space before and after the paragraph.

HTML Code Example:
<p>This is a paragraph of text. The paragraph tag automatically adds spacing before and after the text content, making it appear as a distinct block of text.</p>

<p>This is another paragraph. Each paragraph tag creates a separate text block, making content easier to read and structure.</p>
How it Renders in Browser:

This is a paragraph of text. The paragraph tag automatically adds spacing before and after the text content, making it appear as a distinct block of text.

This is another paragraph. Each paragraph tag creates a separate text block, making content easier to read and structure.

Key Characteristics:

  • Block-level element (starts on a new line)
  • Adds vertical margins automatically
  • Ideal for regular text content
  • Cannot preserve whitespace and line breaks exactly as written

Use of <pre> Tag

<pre> Tag: The preformatted tag preserves both spaces and line breaks exactly as they appear in the HTML code. It's commonly used for displaying code, poetry, or ASCII art.

HTML Code Example:
<pre>
This text is inside a <pre> tag.
It preserves all spaces exactly.

Line breaks are also preserved,
making it perfect for:
* Code snippets
* Poetry
* ASCII art
* Text formatting
</pre>
How it Renders in Browser:
This text is inside a <pre> tag. It preserves all spaces exactly. Line breaks are also preserved, making it perfect for: * Code snippets * Poetry * ASCII art * Text formatting

Key Characteristics:

  • Preserves whitespace and line breaks exactly
  • Uses a monospace font by default
  • Block-level element
  • Ideal for code samples, poetry, formatted text

Comparison: While the <p> tag collapses multiple spaces and line breaks, the <pre> tag preserves the exact formatting.
