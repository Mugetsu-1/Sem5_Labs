Use of <p> tag

This is a paragraph using the <p> tag. The paragraph tag automatically adds spacing before and after the text, making paragraphs distinct from each other.

This is another paragraph. Multiple <p> tags create separate paragraphs with automatic spacing between them.

Use of <pre> tag

This text is inside a <pre> tag.
It preserves    all   spaces    exactly as written.

Line breaks are also kept,
making it perfect for:

    * Code examples
    * Formatted text
    * ASCII art
    * Poetry
